IM1 Week 3 (9/16/2014)

Today we're going to use image tags, and some presentational CSS for floating an element to the left or right of something.

We're going to talk about how a browser "flows" through your HTML document, and what these CSS techniques do to change that.

We'll also continue to review the basics we started last week, like block vs. inline display, the CSS Box Model, and more!

    • Personal About Page

      You'll be using all the tags and techniques we've talked about so far, and now get to really “design” something. Take some basic content about yourself — your photo, some brief bio, some links to things you do, and make a page that adheres to the following requirements:

      1. link to a CSS document
      2. use at least one image
      3. use at least one floating element
      4. use at least one list (<ul> or <ol>)
      5. use semantic tags to properly group your content (titles in <h*> tags, paragraphs in <p> tags, and maybe an <article> or similar for grouping them together...)
      6. use at least one <a> (anchor or link) tag — link out to some of your other accounts or pages, etc… (twitter? dribble? etc.)
