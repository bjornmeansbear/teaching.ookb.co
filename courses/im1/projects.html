Interactive Media and Publishing 1

Fall 2014, Brown 308

Tuesdays, 4pm–10pm

Projects

  • Love/Hate. (2 examples)

    Bring in 2 websites (or apps) — one a website that you love, the other a website that you hate. Be prepared to explain your choices.

  • The Wonderful world of semantic elements (hierarchy!)

    take some basic content (like off a wikipedia page or similar) and code it up by hand using proper, semantic tags we’ve been learning about in class. We’ll review each other’s code for cleanliness and semantic-correctness …

  • Personal "about" page?

    • picture, headlines, lists of stuff, and some paragraphs of text…
    • work with a few basic HTML elements, and then get into some good CSS territory with floating and positioning stuff
    • use at least one image tag, and one CSS image replacement trick

  • The Long Scroll

    (ids, position tricks…, working with content)

    • use IDs AND anchor tags to effectively navigate a long page…
    • Continue to advance CSS comfort with positioning, z-index, etc.

  • HTML/CSS poster

    • (fancy css3 stuff, @font-face…)
    • Use some new(ish) CSS for using real type on the web as well as rotating, animating, etc. of simple objects.
    • Positioning. -- relative, absolute, fixed, & static
    • do something that requires a lot of placing of things…

  • Putting it All Together

    Final project where each student can pick a project of their choosing to put what we’ve learned to use.

In Class Exercises

  • Wk1

    • Hello World.

      first thing, first day go through just the min code to get a page live in the browser

    • Hello World 2.

      first day, but explain what everything in that document means, and add a few more tags (just divs and ps and a few things)

  • Wk2

    • Hello World 3.

      rehash stuff from last week, explain what everything in that document means again, and add a few more tags, like <a> and <ul>, also discuss HTML5 tags more generally...

    • Play around with markdown as a way to learn hierarchy/tagging rules

  • Wk3

    • Hello images

      Redo everything from last week (wk2), but this time include a CSS file and some images, and even more elements!
